Detailed Description

Concrete class for the operator of conditional expectation with respect to gaussian distribution.

This is the concrete class for numerical implementations of the operator of conditional expectation with respect to gaussian distribution. It is constructed by a dynamically allocated implementation of the interface class IGaussRollback.

Constructs a numerical engine that computes the conditional expectation with respect to the gaussian distribution with mean 0 and variance dVar on the grid with step dH and size iSize.

Parameters
pNewPA pointer to a new implementation of the interface class IGaussRollback. The virtual constructor
pNewP->newObject(iSize, dH, dVar)
is called immediately and the result is stored inside of *this.
iSizeThe number of points on the grid.
dHThe distance between the points on the grid.
dVarThe variance of the gaussian distribution.

Rollback operator that also computes the first derivatives with respect to the state variable.

The first derivatives are computed using the integration by parts formula. In total, rollback runs twice.
